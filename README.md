# ClickDesign1.2 (Web版)

## 概要
ClickDesign1.2は、プロフェッショナルな品質のサムネイル画像を自動生成する高機能ツールです。
ダークモードを採用したモダンなインターフェースで、直感的に操作できます。
本バージョンは拡張機能ではなく、Xサーバー等のWebサーバーにアップロードして動作する形式（Webアプリケーション版）です。

## サーバーへの設置（インストール）方法
1. お使いのサーバー（Xサーバーなど）の `public_html` 等の公開用フィルダに、この `clickdesign1.2` フォルダを丸ごとアップロードします。
2. （※案内用の巨大な動画ファイルが存在する場合は、容量圧迫を防ぐためアップロードから除外することをおすすめします。）
3. アップロード完了後、ブラウザで該当のURL（例: `https://あなたのドメイン.com/clickdesign1.2/index.html`）にアクセスすればツールが起動します。

## 使い方
1. 設定したURLにアクセスしてツールを開きます。
2. （任意）専用の「ブックマークレット」をご自身のブラウザに登録しておくと、別ページのテキストを簡単にこのツールに送ることができます。
3. **左サイドバー**でタイトルやページ内容を入力し、参考画像をアップロードします。
4. **メインエリア**でデザインテンプレートを選択し、「生成」ボタンをクリックします。
5. 生成された画像はダウンロードまたはコピーできます。

## デザインの追加について
**重要**: `design` フォルダに手動でフォルダやファイルを追加しても、ツール上には自動的に反映されいません。
新しいデザイン（ヘッダーやサムネイルテンプレートなど）を追加する場合は、以下の手順で行ってください：

1. 追加したいデザインのフォルダを **ZIPファイル** に圧縮します（または `design.json` 単体でも可）。
2. ツールの「インポート」ボタンをクリックします。
3. 作成したZIPファイル（またはJSONファイル）を選択します。

## 設定
- **API設定**: 左上の歯車アイコンからFAL APIキーを設定してください。（一度設定するとブラウザに保存され、次回以降も自動で読み込まれます）
- **デザイン編集**: メインエリア右上の「グループを編集」ボタンから、デザイングループの管理が可能です。
